# Thai Logittory 

## Feature

- ` up to date ` Lottory
- ` history ` list lottoty

## Api
The API is based on HTTPS requests and JSON responses. The stable HTTPS endpoint for the latest version is: ` https://lotto.api.rayriffy.com `

## Request
| Api            | Method |
|----------------|--------|
| /latest        | Get    |
| /list/[:page?] | Get    |
| /lottery/[:id] | Get    |
